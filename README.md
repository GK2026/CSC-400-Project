Project Description -The objective of the project is to create an educational web app to assist students in Prof. Reeves’ classes to understand and complete assignments dealing with finding correlations in real-world development data   

Team Members - 
Evan Wilson: Health Emergency (Helped with SRS and User Guide)  
Grace Kirby:Frontend, UI/UX 
Amir Thompson: Backend,API Integration, Pre-Processing and Sorting data 
Oleksandr Stelmakh:Instructor Features, Server intgration 

Technologies Used - 
Frontend: HTML, CSS, JS, and Chart.js 
Server: Hosted on SCSU server
Backend: Python, FastAPI 

Installation Instructions - 
1. Get server access from ITS
2. Download the VPN from SCSUs website
3. Login to the VPN using your SCSU credentals
4. Use the terminal to log into the server
5. Access the application from chrome 

Running the Application - Use a school computer and put in the URL to start it. Then Explore dada in the way that you want. After you move onto the correlation guide, then you download the data so that it in a Excell/Sheets file, compute correlation, Finally check you correlation once your done.  

Deployment - It is deployed on a SCSU server
