Find the Means

Add up all X values and divide by the number of rows to get X̄. Do the same for Y to get Ȳ. Round to 3 decimal places.

X̄ = (X₁ + X₂ + ... + Xₙ) / n
Ȳ = (Y₁ + Y₂ + ... + Yₙ) / n
